Documentation for a HTML theme by Friendly Themes

01 - Overview

Firstly, allow both of us (Andy and Rich), to say thank you for purchasing one of our items. We have put an awful lot of effort into making this the best in the world. This help document covers pretty much everything you'll need to know about getting up and running.

So, without further ado, let's take a look at what you've bought, what each folder does and what you should do with them.

When you downloaded your zip package and found this help documentation, you will have noticed that there were several other folders. The "PSD" folder contains all of the photoshop files that you may need if you wish to adjust the graphics we've used on this template.

You'll also see a folder called 'the-theme'. In there, unsurprisingly, you'll find all of the page layouts, css, images etc. We keep all of our images, javascript etc. in a 'theme_assets' folder, to keep all things neat and tidy. On top of that, we have a 'template_parts' folder which allows you to make custom templates with ease.

Note: We use Mac OS X when we take our screenshots and screencasts, so things might look slightly different for you, but the way you do things will be exactly the same.

OK, so, now you know what each item is and does, let's move on to installing your theme.

Info

Your Venus template comes with a PHP file called ajax-load.php. This file is used to load in content when the user clicks on the 'plus' icon. If your server is incapable of (or if you don't want to use) PHP files, then you should simply not include that particular section of the header elements (that way the plus icon won't show up)

We've included LESS files which, if you use them as part of your development process then you're free to use. If you're unsure what these are, you are free to ignore them.

We've designed this series of templates to be literally a case for you to copy and paste and edit the content. But there's some things which you may need to know.

For the main menu, if you give one of the top-level li elements a class of current-menu-item then that item will have the green 'active' indicator at the top.

If you wish to change the background of the 'title and search bar' strip on the inner pages, you'll need to edit line 2315 of style.css, alternatively, you can replace sliderbgxy.jpg in /theme_assets/images/

Likewise with the breadcrumb bar, you can edit line 2320 in style.css or change breadcrumb-bg.jpg in /theme_assets/images/

For lots of items, such as 'call to action' buttons and the social icons in the footer, we use a cool little jQuery plugin called tipTip. If you want to use this for your own elements, simply give them a class of "tip" and a title attribute of whatever you want the tip to show when it is hovered, e.g.:

<a href="google.com" title="This will appear in the tip" class="tip">Link Text</a>

We've included lots of 'snippets' for you to use (see the template_parts/ folder) - they'll allow you to add tabs, accordions, buttons, toggles, banners etc. really easily. At the top of each file is a simple bit of help, too, but the idea is you can simply copy and paste.
